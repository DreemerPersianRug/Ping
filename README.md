# Ping
This is very simple version ping program a network utility used to test the reachability and response time of a remote host on a network. It sends ICMP Echo Request packets to the destination host and waits for ICMP Echo Reply packets in response. By measuring the round-trip time (RTT).

# Assembly 
Run "make" in main directory.

# Start
Run "sudo .\ping \<host> \<timeout>" in main directory.
